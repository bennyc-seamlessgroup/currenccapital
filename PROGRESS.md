## Conventions
- Section heading = `.section-heading.centered` ‚Üí `<h2>` + optional `.section-subtitle`
  (centered, gold bar above). NO eyebrows (removed ‚Äî big title carries it).
- Animated bg: add `<div class="anim-bg" aria-hidden="true"></div>` as first child of a
  positioned section; dark variant default, `.anim-bg.light` for light sections.
- Cards link out with `.card-link`; forms compose mailto / show inline success note.

## Screenshot gotcha
Preview screenshots go blank when `scrollY` is large / page is very tall. To check a
deep section: temporarily `display:none` the sections above it via preview_eval, or set
`document.body.style.transform='translateY(-Npx)'`, screenshot, then restore + reload.
